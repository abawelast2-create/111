تعرض لحظياً:

## إدارة الموظفين

### إضافة موظف جديد

1. اضغط **"➕ إضافة موظف"**
2. أدخل:
   - **الاسم الكامل** ← يظهر على صفحة الحضور
   - **المسمى الوظيفي** ← يظهر تحت الاسم
   - **PIN** ← رقم سري فريد (للرجوع إليه مستقبلاً)
   - **الفرع** ← اختر الفرع التابع له الموظف (يحدد نطاقه الجغرافي)
   - **رقم الواتساب** (اختياري) ← بصيغة دولية مثل `966501234567`
3. اضغط **"حفظ"** ← يُولّد الرابط الفريد تلقائياً

> **ملاحظة:** إذا اختير فرع للموظف، سيستخدم النظام إحداثيات الفرع ونصف قطره للتحقق من موقعه، بدلاً من الإعدادات العامة.

### إرسال الرابط للموظف

- **مع رقم واتساب:** اضغط زر **"💬 واتساب"** ← يفتح رسالة جاهزة
- **بدون رقم:** اضغط **"📋 نسخ"** ← انسخ الرابط وأرسله يدوياً

### إجراءات أخرى على الموظف

| الزر | الوظيفة |
|------|---------|
| ✏️ | تعديل الاسم والوظيفة والهاتف |
| 🔄 | إعادة توليد رابط جديد (الرابط القديم يُلغى) |
| 🔕 / 🔔 | تعطيل / تفعيل الموظف |
| 🔒 ربط صارم | تفعيل الربط الصارم — يمنع أي جهاز مختلف |
| 👁️ ربط مراقبة | تفعيل المراقبة الصامتة — يسمح لكن يُسجّل التلاعب |
| 📱🔓 | إعادة تعيين بصمة الجهاز ← يُحرر الرابط من جهاز محدد |
| 🗑️ | أرشفة الموظف (حذف ناعم — يمكن استعادته) |

### الإجراءات الجماعية (أعلى الصفحة)

| الزر | الوظيفة |
|------|--------|
| **تجديد جميع الروابط** | يولّد روابط جديدة لكل الموظفين النشطين |
| **إرسال الكل واتساب** | رسالة واحدة بكل الروابط مرتبة حسب الفرع |
| **🔒 ربط صارم للجميع** | تفعيل الربط الصارم لجميع الموظفين دفعة واحدة |
| **👁️ مراقبة للجميع** | تفعيل المراقبة الصامتة لجميع الموظفين |

### عمود "الجهاز"

| الأيقونة | المعنى |
|----------|--------|
| 🔒 | ربط صارم — الرابط مربوط بجهاز ويُحظر أي جهاز آخر |
| 👁️ | مراقبة صامتة — مربوط لكن يسمح ويُسجّل التلاعب |
| ⏳ ينتظر | وضع الربط مفعّل — سيُربط عند الدخول التالي |
| 🔓 حر | الرابط حر — لا يحتاج ربط جهاز |
| 📱 اسم الموظف | مالك الجهاز — الموظف صاحب أعلى استخدام لنفس الجهاز |

## صفحات إضافية

### بروفايل الموظف (`/admin/employee-profile.php`)
- عرض وإدارة بيانات الموظف الكاملة مع صورة البروفايل
- إدارة مجموعات الوثائق (هوية، رخصة، جواز...) مع تواريخ الانتهاء
- رفع/حذف صور ووثائق (دعم JPG/PNG/WebP/PDF)
- عرض تنبيه للوثائق المنتهية أو القريبة من الانتهاء

### متابعة انتهاء الوثائق (`/admin/documents-expiry.php`)
- قائمة بجميع مجموعات الوثائق المنتهية أو القريبة من الانتهاء
- تصفية حسب الحالة والفرع
- تنبيهات بصرية بالألوان

### تعديل الفرع (`/admin/branch-edit.php`)
- تعديل بيانات الفرع (الاسم، الإحداثيات، نصف القطر)
- إدارة الورديات (حتى 3 ورديات لكل فرع)
- إعدادات الدوام الإضافي لكل فرع

### التقارير البيانية (`/admin/report-charts.php`)
- رسوم بيانية تفاعلية لإحصائيات الحضور
- مقارنة بين الفروع والموظفين
- تقارير الأداء والتأخر

### تقرير التأخر (`/admin/late-report.php`)
- يُظهر الموظفين الذين تأخروا عن وقت الدخول
- قابل للتصفية حسب التاريخ والفرع

### التقرير اليومي (`/admin/report-daily.php`)
- ملخص يومي شامل للحضور والانصراف والغياب

### حالات التلاعب (`/admin/tampering.php`)
- جميع حالات التلاعب المُكتشفة (جهاز مختلف، تسجيل بالنيابة...)
- يُظهر الخطورة والتفاصيل ومالك الجهاز

### البلاغات السرية (`/admin/secret-reports.php`)
- البلاغات المُرسلة من الموظفين عبر زر البلاغ العائم
- تشمل: نص + صور + تسجيل صوتي (مع مؤثرات)
- حالات: جديد / تمت المراجعة / قيد المعالجة / مُعالَج / مرفوض / مؤرشف
- ملاحظات المشرف + تاريخ المراجعة

### الإجازات (`/admin/leaves.php`)
- إضافة إجازات (سنوية/مرضية/بدون راتب/أخرى)
- عرض وتصفية الإجازات حسب الحالة والتاريخ
- الموافقة أو الرفض مع تسجيل المشرف المسؤول

## الإعدادات

### تبويب الموقع الجغرافي
- **موقع العمل:** دبّن الموقع على الخريطة التفاعلية أو أدخل الإحداثيات يدوياً
- **نصف قطر النطاق:** المسافة المسموحة بالمتر (الحد الأدنى 1 متر)
  - مبنى صغير: 50-100 متر
  - مجمع: 200-500 متر
  - ميدان واسع: 500-1000 متر

### تبويب أوقات الدوام
| الإعداد | الوصف |
|---------|-------|
| بداية الدوام الافتراضية | وقت بداية الدوام الافتراضي (يُستخدم عند إنشاء فرع جديد) |
| نهاية الدوام الافتراضية | وقت نهاية الدوام الافتراضي |

> **ملاحظة:** أوقات الدوام الفعلية تُدار الآن من خلال **نظام الورديات** في صفحة تعديل الفرع. كل فرع يدعم حتى 3 ورديات مستقلة.

### نظام الورديات (جديد)
- كل فرع يدعم حتى **3 ورديات** مع أوقات بداية/نهاية مستقلة
- النظام **يكشف تلقائياً** الوردية النشطة بناءً على الوقت الحالي
- نافذة التسجيل: بداية الوردية − 60 دقيقة حتى نهاية الوردية
- يدعم الورديات التي تعبر منتصف الليل (مثل 23:00 → 07:00)
- إدارة الورديات من: **الفروع** ← تعديل الفرع ← الورديات

### واتساب لكل فرع (جديد)
- زر "📱 واتساب (N)" في كل بطاقة فرع بصفحة الفروع
- يفتح نوافذ واتساب متعددة (كل 800ms) لجميع موظفي الفرع
- الرسالة تتضمن: اسم الموظف + رابط البوابة + رقم PIN

### باركود QR لبوابة الموظف (جديد)
- زر "باركود البوابة" بجانب زر بوابة الحضور في صفحة الموظفين
- يولّد QR Code بدقة عالية مع شعار التطبيق في المنتصف
- أزرار تحميل PNG وطباعة

### تبويب الدوام الإضافي
- تفعيل/تعطيل ميزة الدوام الإضافي
- بعد كم دقيقة من الانصراف يبدأ
- الحد الأدنى للمدة

### تبويب الأمان
- تغيير كلمة مرور المشرف (يشترط كلمة المرور الحالية)

## سيناريوهات شائعة

### ربط جهاز موظف لأول مرة
1. الموظف يفتح رابطه → يمكنه التسجيل بدون ربط جهاز
2. عندما تريد ربطه، اختر من القائمة:
   - **🔒 ربط صارم** ← لمنع أي جهاز آخر
   - **👁️ ربط مراقبة** ← للسماح مع تسجيل التلاعب
3. الموظف يفتح الرابط على جهازه ← يُربط الجهاز تلقائياً
4. بعد الربط:
   - **صارم:** لن يعمل الرابط على أي جهاز آخر
   - **مراقبة:** يعمل من أي جهاز لكن التلاعب يُسجّل

> **ملاحظة:** لا يحدث ربط تلقائي — المشرف يتحكم بالكامل.

### الموظف غيّر هاتفه
1. ابحث عن الموظف في قائمة الموظفين
2. اضغط زر **📱🔓** (فك ربط الجهاز)
3. إذا أردت ربط الجهاز الجديد، اختر **🔒 ربط صارم** أو **👁️ ربط مراقبة**
4. الموظف يفتح رابطه على الجهاز الجديد → يُربط تلقائياً

### تجديد وإرسال جميع الروابط
1. اضغط **"تجديد جميع الروابط"** ← يولّد روابط جديدة للجميع
2. اضغط **"إرسال الكل واتساب"** ← يفتح واتساب برسالة شاملة
3. الرسالة تحتوي كل الروابط مرتبة حسب الفرع

### الموظف فقد رابطه
1. من قائمة الموظفين اضغط **🔄** لتوليد رابط جديد
2. أرسل الرابط الجديد عبر واتساب
3. الرابط القديم يُلغى تلقائياً

### فتح الرابط من جهاز مختلف (اشتباه بالتشارك)
- صفحة الموظف ستعرض شاشة القفل 🔒 (إذا كان الجهاز مربوطاً)
- إذا لم يكن مربوطاً، يمكنه الدخول من أي جهاز
- لا يمكنه التسجيل من جهاز آخر مربوط حتى تتدخّل أنت
