## تحديثات الإنتاج (مارس 2026)

### الربط والبنية التحتية
- الدومين الرسمي الحالي: https://mycorner.site
- تم تفعيل HTTPS بشهادة Let's Encrypt مع تحويل تلقائي من HTTP
- تم تضمين `StartSession` و `EncryptCookies` في مجموعة `api` middleware

### إصلاحات حرجة
- **رسائل Flash:** تم تصحيح مفاتيح الجلسة في القالب (`flash_success` ← `success`، `flash_error` ← `error`) لعرض رسائل النجاح والخطأ
- **زر تسجيل الخروج:** تم تحويله من `<a href>` (GET) إلى `<form method="POST">` ليتوافق مع المسار المعرّف كـ POST
- **رموز استرداد 2FA:** تم إضافة `decrypt()` عند التحقق من الرموز المشفرة بـ `encrypt()`
- **حذف بيانات اعتماد مكشوفة:** تم حذف 25+ ملف Python وملفات MD تحتوي كلمات مرور من السيرفر

### إصلاحات أمنية
- **شارة الإشعارات:** تم تصحيح `notifiable_type` من `'admin'` إلى `Admin::class` (`App\Models\Admin`)
- **Mass Assignment:** تم استبدال `$request->all()` بـ `$request->validated()` في `BranchController`
- **BackupService:** تم استبدال `addslashes()` بـ `PDO::quote()` لحماية SQL
- **Rate Limiting:** تم إضافة `throttle:30,1` لمسار `/api/get-employee`
- **CORS:** تم تقييد `allowed_origins` من `['*']` إلى `['https://mycorner.site', 'https://www.mycorner.site']`

### إصلاحات الواجهة
- رسالة انتهاء الجلسة تظهر الآن في صفحة تسجيل الدخول
- تم تنظيف المسارات المكررة في `routes/web.php`
- تم تحميل ملفات خط Tajawal (10 ملفات woff2) لإصلاح المظهر العربي
