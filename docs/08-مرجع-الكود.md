### GeofenceService — التحقق الجغرافي

```php
// حساب المسافة بين نقطتين (صيغة Haversine)
GeofenceService::calculateDistance(
    lat1: float, lon1: float,  // موقع الموظف
    lat2: float, lon2: float   // موقع الفرع
): float
// Returns: المسافة بالمتر

// Haversine Formula:
// a = sin²(Δlat/2) + cos(lat1) × cos(lat2) × sin²(Δlon/2)
// c = 2 × asin(√a)
// d = R × c  (R = 6,371,000 meters)

// فحص: هل الموظف داخل الجيوفينس؟
GeofenceService::isWithinGeofence(
    empLat: float,
    empLon: float,
    branchId: ?int
): array
// Returns: ['allowed' => bool, 'distance' => float, 'radius' => int, 'message' => string]

// أولوية الفحص:
// 1. إذا branchId موجود → إحداثيات الفرع
// 2. إذا لا → الإعدادات العامة (جدول settings)
// 3. إذا لم يُحدد موقع عمل → allowed = true (مسموح دائماً)
```

## حساب بصمة الجهاز (JavaScript — واجهة الموظف)

```javascript
async function getFingerprint() {
    const parts = [
        navigator.userAgent,
        screen.width + 'x' + screen.height + 'x' + screen.colorDepth,
        Intl.DateTimeFormat().resolvedOptions().timeZone,
        navigator.language,
        navigator.platform || ''
    ];
    
    // Canvas fingerprint — بصمة رسومية فريدة
    const c = document.createElement('canvas');
    const x = c.getContext('2d');
    x.font = '14px Arial';
    x.fillText('fp', 2, 2);
    parts.push(c.toDataURL().slice(-50));

    const raw = parts.join('|');
    const buf = await crypto.subtle.digest(
        'SHA-256', new TextEncoder().encode(raw)
    );
    return Array.from(new Uint8Array(buf))
        .map(b => b.toString(16).padStart(2, '0')).join('');
}
```

**المكونات:**
1. `userAgent` — نظام التشغيل والمتصفح
2. `screen.width × height × colorDepth` — مواصفات الشاشة
3. `Intl timezone` — المنطقة الزمنية
4. `navigator.language` — لغة الجهاز
5. `navigator.platform` — نظام التشغيل
6. `Canvas fingerprint` — بصمة رسومية فريدة للـ GPU

**النتيجة:** SHA-256 hash (64 حرف hex)
