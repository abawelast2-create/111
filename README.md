# نظام إدارة الحضور والانصراف — Sarh.io

> نظام حضور وانصراف ذكي يعتمد على الموقع الجغرافي (GPS) وربط الأجهزة، مبني بـ Laravel 11.

**الموقع:** https://sarh.io

---

## الميزات الرئيسية

- **تسجيل حضور بالموقع الجغرافي** — يتحقق أن الموظف داخل نطاق مقر العمل
- **دعم ورديتين** — وردية صباحية ومسائية بمواعيد مستقلة
- **ربط الجهاز** — 3 أوضاع (حر / صارم / مراقبة) لمنع التلاعب
- **لوحة تحكم لحظية** — تحديث تلقائي كل 15 ثانية
- **تقارير متعددة** — تأخر، رسوم بيانية، تصدير CSV
- **إدارة فروع** — كل فرع بإعداداته الجغرافية والزمنية
- **العمل الإضافي** — تسجيل وتتبع ساعات العمل الإضافي
- **إجازات** — طلب وموافقة ورفض
- **بلاغات سرية** — يقدمها الموظفون مع صور
- **كشف التلاعب** — تنبيهات فورية عند تغيير الجهاز
- **روابط واتساب** — إرسال روابط الحضور تلقائياً
- **سجل مراجعة** — تسجيل كل إجراءات المدير

---

## المتطلبات

- PHP 8.2+
- MariaDB / MySQL 8.0+
- Composer 2.x

---

## التثبيت السريع

```bash
# 1. تثبيت التبعيات
composer install --no-dev --optimize-autoloader

# 2. نسخ ملف البيئة
cp .env.example .env

# 3. تعديل إعدادات قاعدة البيانات في .env

# 4. توليد مفتاح التطبيق
php artisan key:generate

# 5. تنفيذ الهجرات
php artisan migrate --seed

# 6. تحسين الأداء
php artisan optimize
```
